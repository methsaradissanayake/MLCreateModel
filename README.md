# ML Create Model Project

This is a clean machine learning project structure.

## Project Structure

- `data/`: Placeholder for raw and processed datasets.
- `notebooks/`: Jupyter notebooks for EDA and experimentation.
- `src/`: Python source code for the ML pipeline.
- `models/`: Saved model files (e.g., .pkl, .h5).
- `docs/`: Documentation and reports.

## Getting Started

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
